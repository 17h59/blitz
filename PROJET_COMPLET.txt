NOTES POUR LE RAPPORT NSI
==========================

Technologies utilisées :
- Backend : PHP 8.x + SQLite3
- Frontend : Vanilla JavaScript (pas de framework)
- CSS : Variables CSS + Flexbox/Grid
- Real-time : SSE (Server-Sent Events) + Polling
- Markdown : Parsing custom
- Sécurité : password_hash, htmlspecialchars, CSRF tokens

Temps de développement :
- Base (chat + auth) : ~3h
- DMs et profils : ~2h
- Forum complet : ~2h
- Total : ~7h en collaboration avec Claude AI

Fichiers : 21 fichiers PHP + 4 fichiers JS/CSS
Lignes de code : ~3500 lignes
Tables BDD : 15 tables avec CASCADE
